# capacitor-plugin-audiotoggle-bluetooth

Capacitor plugin for switching between speaker, earpiece and bluetooth headset when playing audio.

## Install

```bash
npm install capacitor-plugin-audiotoggle-bluetooth
npx cap sync
```
