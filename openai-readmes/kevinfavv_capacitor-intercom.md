# @kevinfav/capacitor-intercom 
Fork from [@sencrop/capacitor-intercom](https://github.com/sencrop/capacitor-intercom)

Intercom Plugin for Capacitor 6+

## Install

```bash
npm install @kevinfav/capacitor-intercom
npx cap sync
```
