# Using @globy-app/capacitor-sumup Package Tutorial

To integrate SumUp payments into your Capacitor app, you can make use of the @globy-app/capacitor-sumup package. Follow the steps below to get started:

1. **Installation:**
   - Install the package using npm:
     ```
     npm install @globy-app/capacitor-sumup
     ```

2. **Configuration:**
   - Configure the SumUp credentials in your Capacitor configuration file.

3. **Usage:**
   - Use the provided methods in the package to initiate payments, authorize payments, and handle payment responses.

4. **Error Handling:**
   - Make sure to handle errors that might occur during the payment process.

5. **Testing:**
   - Test the integration thoroughly to ensure a seamless payment experience for your users.

By following these steps, you can successfully integrate SumUp payments into your Capacitor app using the @globy-app/capacitor-sumup package.
