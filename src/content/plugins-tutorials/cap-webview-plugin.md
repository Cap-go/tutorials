# Using cap-webview-plugin Package Tutorial

The cap-webview-plugin allows you to display websites inside a Capacitor App. Here's a step-by-step guide on how to use this plugin in your app:

## Installation

To install the cap-webview-plugin package, run the following commands:

```bash
npm install cap-webview-plugin
npx cap sync
```

## Usage

Once the package is installed, you can start using it to display web content within your Capacitor app. Make sure to refer to the API documentation to understand how to interact with the plugin effectively.

For more detailed information and examples, please check the official documentation provided for the cap-webview-plugin package.

Happy coding!
