# Using Capacitor Emarsys SDK Package Tutorial

This tutorial will guide you on how to use the @rollershop/capacitor-emarsys-sdk package. Below are the steps to get started:

## Installation

To install the package, run the following command in your terminal:

```bash
npm install @rollershop/capacitor-emarsys-sdk
npx cap sync
```

## Setup
### Initialize SDK

To initialize the SDK, follow these steps:

1. Import the package into your project.
2. Use the available options for initializing the SDK, as outlined in the documentation.

For more detailed information on the available SDK options and methods, refer to the [official documentation](https://help.emarsys.com/hc/en-us).

---

Remember to check the official documentation for any updates or additional features.
