# Using star-micronics-capacitor package

To use the `star-micronics-capacitor` package in your Capacitor project, follow these steps:

1. Install the package by running `npm install star-micronics-capacitor`.
2. Import the package in your Capacitor project.
3. Follow the package documentation to interact with Star Micronics devices.

That's it! You are now ready to use the `star-micronics-capacitor` package in your Capacitor project.
