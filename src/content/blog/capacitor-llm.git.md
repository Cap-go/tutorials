## Getting Started with @capgo/capacitor-llm Package

To get started with the @capgo/capacitor-llm package, follow these steps:

### Step 1: Installation

1. Install the package using npm:
   ```bash
   npm install @capgo/capacitor-llm
   ```

2. Sync your Capacitor project:
   ```bash
   npx cap sync
   ```

### Step 2: Using the Package

After installation, you can use the @capgo/capacitor-llm package in your project by following these guidelines:

1. Import the package in your code:
   ```typescript
   import 'capacitor-llm';
   ```

2. Utilize the functionalities provided by the package in your Capacitor project.

### Step 3: Build and Run

Once you have integrated the package into your project, build and run your Capacitor app to see the changes in action:

1. Build your project:
   ```bash
   npm run build
   ```

2. Copy the project to your native platform:
   ```bash
   npx cap copy
   ```

3. Add any specific platforms you are targeting:
   ```bash
   npx cap add <platform>
   ```

4. Run your project:
   ```bash
   npx cap open <platform>
   ```

By following these steps, you can successfully integrate and use the @capgo/capacitor-llm package in your Capacitor project.
