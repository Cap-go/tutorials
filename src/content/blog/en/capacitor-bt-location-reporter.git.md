# Using @tovaz/capacitor-bt-location-reporter Package

To get started with the @tovaz/capacitor-bt-location-reporter package, follow these steps:

1. Install the package in your Capacitor project:
   
   ```bash
   $ npm install @tovaz/capacitor-bt-location-reporter
   ```

2. Import the package in your project:

   ```javascript
   import BtLocationReporter from '@tovaz/capacitor-bt-location-reporter';
   ```

3. Initialize the Bluetooth Location Reporter:

   ```javascript
   const btLocationReporter = new BtLocationReporter();
   ```

4. Start using the package to report Bluetooth locations in your app.

Remember to refer to the package documentation for detailed API reference and guidance on advanced usage.

Happy coding!
