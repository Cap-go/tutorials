# Using @capacitor-community/device-check

In this tutorial, we will guide you through the process of incorporating the @capacitor-community/device-check package into your Capacitor app.

## Step 1: Installation

First, install the package by running the following command in your project directory:

```bash
npm install @capacitor-community/device-check
```

## Step 2: Importing the Plugin

Next, import the DeviceCheck plugin into your project. Ensure that you import it in the necessary files where you plan to use it.

```typescript
import '@capacitor-community/device-check';
```

## Step 3: Functionality

With the plugin installed and imported, you can now utilize the various functionalities it provides. Make sure to refer to the official documentation for the @capacitor-community/device-check package to understand how to use each feature effectively.

That's it! You've successfully integrated the @capacitor-community/device-check package into your Capacitor app. Happy coding!
