# Using Capacitor-Hugin-Gmp3 Package

To start using the Capacitor-Hugin-Gmp3 package, follow these steps:

1. Install the package by running `npm install capacitor-hugin-gmp3`.
2. Import the package in your Capacitor project.
3. Use the provided functions and methods to work with audio files.

Make sure to refer to the official documentation for detailed usage instructions and examples.
