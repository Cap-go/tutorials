# Using Capacitor-Widget-Bridge Package

To begin using the Capacitor-Widget-Bridge package in your project, follow these steps:

1. Install the package by running `npm install capacitor-widget-bridge` in your project directory.
   
2. Import the necessary components in your project files:
   ```javascript
   import { WidgetBridge } from 'capacitor-widget-bridge';
   ```
   
3. Initialize the WidgetBridge in your project:
   ```javascript
   const widgetBridge = new WidgetBridge();
   ```
   
4. Start using the capabilities provided by the WidgetBridge in your Capacitor project.

This tutorial will guide you through the process of integrating the Capacitor-Widget-Bridge package seamlessly into your Capacitor projects.
