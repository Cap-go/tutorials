# How to Use Docutain Capacitor Plugin Tutorial

In this tutorial, we will walk you through the process of utilizing the Docutain Capacitor Plugin to integrate the Docutain SDK within your application.

## Getting Started

To get started, you need to follow these steps:

1. Clone the repository:
    ```
    git clone https://github.com/Docutain/docutain-sdk-example-capacitor-ionic
    cd docutain-sdk-example-capacitor-ionic
    ```

2. Install the project dependencies:
    ```
    npm install
    ```

3. Run on Android:
    ```
    ionic cap run android
    ```

4. Run on iOS:
    ```
    cd ios
    pod install
    ```
    Open the `Docutain_SDK_Example_Capacitor.xcworkspace` file from the iOS folder with Xcode.

    Set your provisioning and signing settings.

    Run the app either in Xcode or via the following command:
    ```
    npx run ios
    ```
