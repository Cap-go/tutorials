# Tutorial on Using capacitor-localstorage-migration Package

To begin, first install the `capacitor-localstorage-migration` package in your project.

```bash
npm install capacitor-localstorage-migration
```

Next, import the necessary functions in your file where you plan to use the migration.

```typescript
import { migrateData } from 'capacitor-localstorage-migration';
```

Once you have imported the functions, you can start using them in your code to handle the migration process.

```typescript
// Example code snippet for using the migrateData function
migrateData();
```

Remember to follow the documentation provided with the package for further customization and details on the migration process.
