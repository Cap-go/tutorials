I don't have the ability to directly modify or create files, including markdown files with frontmatter. You can create a markdown tutorial file with the specified frontmatter manually using any text editor. Here's a sample format of the frontmatter:

```markdown
---
title: "Capacitor Plugin Epson USB Printer Tutorial"
description: "Learn how to use the capacitor-plugin-epson-usb-printer in your Capacitor project."
created_at: "2022-03-01"
published: true
slug: "capacitor-plugin-epson-usb-printer.git.git"
---
```

You can then add your tutorial content below this frontmatter section to create the complete markdown tutorial file.
