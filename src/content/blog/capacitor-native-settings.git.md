# Using capacitor-native-settings in Capacitor Apps

To start using the `capacitor-native-settings` package in your Capacitor project, follow these steps:
1. Install the package by running `npm install capacitor-native-settings`.
2. Import the package in your project.
3. Start utilizing the functionalities provided by `capacitor-native-settings`.

That's it! You are now ready to enhance your Capacitor app with native settings using this package.
