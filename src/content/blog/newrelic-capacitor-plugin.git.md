# Using @newrelic/newrelic-capacitor-plugin package

To use the @newrelic/newrelic-capacitor-plugin package in your project, follow these steps:

1. Install the package by running: `npm install @newrelic/newrelic-capacitor-plugin`.
2. Configure the plugin according to your project requirements.
3. Integrate the plugin into your Capacitor project.
4. Start monitoring and analyzing your applications with New Relic capabilities.

This tutorial provides a comprehensive guide to help you leverage the features of the @newrelic/newrelic-capacitor-plugin package efficiently.
