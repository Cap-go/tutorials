## Introduction

In this tutorial, we will walk through the steps of integrating and using the @capacitor-community/native-audio package in your Capacitor project for playing sounds natively. This plugin provides a way to leverage native audio capabilities in your cross-platform apps built with Capacitor. Let's get started!


## Installation

You can install the @capacitor-community/native-audio package using npm or yarn. Run the following command based on your preferred package manager:

```bash
npm install @capacitor-community/native-audio
```

or

```bash
yarn add @capacitor-community/native-audio
```

After installing the package, make sure to sync the native files by running:

```bash
npx cap sync
```


## Configuration

There is no additional configuration required for this plugin. It's ready to use once installed in your Capacitor project.


## Usage

The plugin provides methods for playing sounds natively. You can use these methods in your project to leverage the native audio capabilities. 

For more details on the specific methods supported by the @capacitor-community/native-audio package, refer to the [official documentation](https://www.npmjs.com/package/@capacitor-community/native-audio).
