# Using @leonardojc/capacitor-ioboard package

This blog post will guide you through the process of installing, configuring, and using the @leonardojc/capacitor-ioboard package in your Capacitor project.

### Installation

To get started, install the package using npm:

```bash
npm install @leonardojc/capacitor-ioboard
```

### Configuration

Next, make sure to sync the package with Capacitor:

```bash
npx cap sync
```

### Usage

Now that the package is installed and synchronized, you can start using it in your project. Refer to the package's documentation for detailed usage instructions.

That's it! You have successfully installed and configured the @leonardojc/capacitor-ioboard package in your Capacitor project.
