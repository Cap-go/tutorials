# Using @jonz94/capacitor-sim Package

To use the @jonz94/capacitor-sim package in your project, follow these steps:

1. Install the package by running:
   ```bash
   npm install @jonz94/capacitor-sim
   ```

2. Import the package in your code:
   ```typescript
   import { CapacitorSim } from '@jonz94/capacitor-sim';
   ```

3. Initialize the plugin in your code:
   ```typescript
   const sim = new CapacitorSim();
   ```

4. Use the plugin's functionalities in your project.

That's it! You have successfully integrated the @jonz94/capacitor-sim package into your Capacitor project.
