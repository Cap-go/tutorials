# Using capacitor-plugin-health-connect-steps Package Tutorial

In this tutorial, you will learn how to integrate and use the `capacitor-plugin-health-connect-steps` package in your Capacitor project.

## Step 1: Installation

To get started, install the `capacitor-plugin-health-connect-steps` package by running the following command:

```bash
npm install capacitor-plugin-health-connect-steps
```

## Step 2: Initialization

Next, sync the plugin with your Capacitor project using the `npx cap sync` command:

```bash
npx cap sync
```

## Step 3: Using the Plugin

Now that the plugin is installed and synced, you can start using its functionality in your project. Make sure to refer to the plugin documentation for details on how to leverage its features.

That's it! You have successfully integrated the `capacitor-plugin-health-connect-steps` package into your Capacitor project.
